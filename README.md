HTML Form Construction – User Registration & Checkout
Overview

This project implements a comprehensive HTML form designed for user registration and checkout. The form collects essential user, contact, billing, and payment information in a structured and user-friendly format.

Features

The form is divided into multiple sections using <fieldset> for better organization and accessibility.

1. Account Information

Full Name

Username

Email Address

Password

Confirm Password

2. Contact Information

Phone Number

Company Name (Optional)

Website (Optional)

3. Billing Information

Billing Name

Email

Address Line 1

Address Line 2 (Optional)

City

State

Postal Code

Country (Dropdown selection)

4. Payment Information

Card Holder Name

Card Number

Expiry Date (Month/Year)

CVV (Masked using password input)

Implementation Details

Built using HTML5 form elements

Each input field is associated with a <label> for accessibility

<fieldset> and <legend> are used to group related form sections

Appropriate input types such as email, tel, password, and number are used to improve validation and usability

Purpose

This assignment demonstrates the use of structured HTML forms, semantic grouping, and proper input types to create a clear and accessible user data collection form.
